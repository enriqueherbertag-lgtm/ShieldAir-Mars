# Certificaciones - ShieldAir-Mars

Esta carpeta contiene los planes y documentos de certificación para equipos espaciales.

## Contenido planificado

- `nasa_std_3001.md` — cumplimiento de NASA STD-3001 (requisitos para sistemas de soporte vital en vuelos espaciales tripulados).
- `ecss_requirements.md` — requisitos ECSS (Cooperación Europea para Estándares Espaciales).
- `radiation_hardening.md` — plan de endurecimiento radiológico para componentes electrónicos.
- `thermal_vacuum_test.md` — plan de pruebas en vacío térmico (simulación de atmósfera marciana).

## Certificaciones aplicables

| Certificación | Organismo | Ámbito |
|---------------|-----------|--------|
| NASA STD-3001 | NASA | Sistemas de soporte vital para misiones tripuladas |
| ECSS-Q-ST-70 | ECSS | Materiales y procesos para equipos espaciales |
| MIL-STD-810H | DoD (EE.UU.) | Pruebas ambientales (temperatura, vacío, vibración) |
| ISO 14624 | ISO | Seguridad de sistemas de soporte vital |
