# Hardware - ShieldAir-Mars

Esta carpeta contiene los esquemas y diseños mecánicos de la torre ShieldAir-Mars.

## Contenido planificado

- `torre_schematic.pdf` — esquema general de la torre (dimensiones, materiales, flujo de gases).
- `compresores_specs.pdf` — especificaciones de los compresores para captura de CO₂.
- `bioreactor_schematic.pdf` — diseño del reactor de algas (Scenedesmus dimorphus, Chlorella).
- `led_system.pdf` — sistema de iluminación LED para fotosíntesis.
- `heat_exchanger.pdf` — intercambiadores de calor para recuperación térmica.
- `cryogenic_system.pdf` — sistema de licuefacción para producción de O₂ líquido.

## Estado

- Esquemas electrónicos pendientes.
