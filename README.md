# ShieldAir - Marte

[![License: CC BY-NC 4.0](https://img.shields.io/badge/License-CC_BY--NC_4.0-lightgrey.svg)](https://creativecommons.org/licenses/by-nc/4.0/)

**Producción de oxígeno, biomasa y calor desde la atmósfera marciana.**

ShieldAir es una torre modular adaptada de su versión terrestre (originariamente diseñada para limpiar smog urbano). En Marte, convierte el CO₂ atmosférico (95%) en recursos vitales usando energía nuclear, luz LED y agua extraída del hielo marciano.

## ¿Qué produce?

| Recurso | Producción por torre (estimada) | Uso |
|---------|--------------------------------|-----|
| **Oxígeno (O₂)** | 0.8 toneladas/día | Respiración, oxidante para Goliat-Son |
| **Biomasa (algas)** | 15 kg/día | Alimento, bioplásticos, fertilizantes |
| **Calor** | 60–80 kW térmicos | Calefacción, derretimiento de hielo |

## Adaptación de la versión terrestre

| Parámetro | Versión Tierra (original) | Versión Marte (adaptada) |
|-----------|---------------------------|--------------------------|
| **Entrada** | Aire con smog (PM2.5, CO₂) | CO₂ atmosférico (95%) |
| **Proceso** | Algas + luz solar + CO₂ | Algas + luz LED + CO₂ + energía nuclear |
| **Salida** | Aire limpio, calor, biomasa | O₂ líquido, biomasa, calor |
| **Energía** | Calor recuperado + eléctrica | Energía nuclear + paneles solares |

## ¿Cómo funciona?

| Etapa | Proceso | Producto |
|-------|---------|----------|
| **1. Captura** | Compresores aspiran aire marciano (95% CO₂) | CO₂ concentrado |
| **2. Fotosíntesis** | Algas (*Scenedesmus dimorphus*, *Chlorella*) + CO₂ + H₂O + luz LED → O₂ + biomasa | O₂, algas |
| **3. Licuefacción** | Enfriamiento criogénico | O₂ líquido (LOX) |
| **4. Procesamiento de biomasa** | Secado, prensado, fermentación | Proteína, lípidos, bioplásticos |
| **5. Recuperación de calor** | Intercambiadores de calor | Calor para hábitats, derretimiento de hielo |

## Requerimientos por torre

| Recurso | Cantidad diaria | Origen |
|---------|-----------------|--------|
| CO₂ | 1.1 toneladas | Atmósfera marciana |
| Agua | 0.5 toneladas | Hielo subterráneo (extraído) |
| Energía eléctrica | 10–15 kW | Reactor nuclear de Odiseo o paneles solares |
| Luz LED | 8–10 kW | Energía eléctrica |
| Nutrientes | 1 kg | Traídos de Tierra (inicial), luego reciclados |

## Integración con Odiseo y Goliat-Son

| Proyecto | Uso de ShieldAir |
|----------|------------------|
| **Odiseo** | Energía nuclear alimenta las torres; calor recuperado se usa en la base. |
| **Goliat-Son** | Reabastece sus tanques de O₂ con LOX producido por las torres. |
| **Base marciana** | O₂ para respirar, calor para hábitats, biomasa para alimento/plásticos. |

## Escalabilidad

Para la primera base marciana (6 personas), se requieren:
- **2 torres ShieldAir** operando continuamente.
- Producción total: 1.6 toneladas O₂/día (suficiente para respiración + oxidante para Goliat-Son).
- Excedente de biomasa: 30 kg/día (alimento o plásticos).
- Calor: 120–160 kW térmicos (calefacción y derretimiento de hielo).
